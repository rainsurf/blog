Creating This Site
Mar 5, 2017
2 minutes read

Step 0: Inspiration

I’ve always wanted to have my own blog or site. But I never really liked the idea of using a service like Medium or Wordpress or Ghost. I wanted my site to be my “own.”

I wanted to dip my toe in web development ocean with the hope of one day scuba diving in it. But I needed inspiration. The site that inspired me on is linked on the about section. I was browsing HackerNews (the exact article is linked in the ‘About’ section), came across an interesting article about a face-replacing slack bot, and was simultaneously struck by the simple and minimal design of the blog.

And from there it snowballed as I rapidly tried to understand:

  • What is Hugo?
    • Oh, it’s a static web site engine.
  • What is a static website engine?
    • I don’t totally understand yet, but I will.
  • Is Hugo the best static website engine for me?
    • Yes, good enough to start. One of the most simple to use.
  • How can I create my own Hugo site on Azure?
    • Let’s “Google” it. Oh this guy did it using Azure Blob Storage.
  • What is Azure Blog Storage?
    • I don’t totally understand yet, but I will.
  • Ok, Azure Blog Storage has too many compatibility issues wiht Hugo and I don’t want to work through them right now, what else can I use to host?
    • I could just use GitHub Pages, but I want to learn more about Azure. Ok, I will use an Azure Web App.

And… here we are. There are a few is a shit ton of other things that I will need to learn more about, but for now I am satisfied with figuring out the basics and hacking it together.

How I Feel Right Now

Learning and creating something - no matter how small - feels good!
